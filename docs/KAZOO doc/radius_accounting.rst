Radius Accounting
=================

This endpoint provides carrier accounting records.

Request
=======

``GET`` request to ``v2/accounts/{ACCOUNT_ID}/radius_accounts`` with the
following query params:

-  ``other_party``
-  ``billing_party``
-  ``date_from`` - in format DD-MM-YYYY
-  ``date_to`` - in format DD-MM-YYYY
-  ``billing_id``
